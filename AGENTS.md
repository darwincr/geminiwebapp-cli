# AGENTS.md

- For software development tasks, always read `README.md`.
- For general Gemini operation, always read `BROWSER_OPERATION.md`.
- Always use `uv` for Python: `uv run` to execute scripts, `uv sync` to install dependencies, `uv run python` instead of bare `python`.
- After pushing a new version upstream, run `uv tool upgrade geminiwebapp-cli` to update the system-installed CLI at `/Users/darwin/.local/bin/geminiwebapp-cli`.
